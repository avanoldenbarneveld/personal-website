# Personal Website

This is my personal website and blog, built with Next.js. It serves as a place to document my journey in coding, share insights, and showcase my expertise.

## Features
- **Simple Structure**: Includes an About page, Blog, and Contact page.
- **Fast & Modern**: Built with Next.js for optimized performance.
- **Markdown Support**: Blog posts are written in Markdown for easy content management.
- **Deployed on Vercel**: Automatically updates with each GitHub push.

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Hosting**: Vercel
- **Content**: Markdown

## Setup Instructions
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/personal-website.git
   cd personal-website
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view the site.
